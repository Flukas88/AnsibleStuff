AnsibleStuff

Ansible Roles for my cloud. At the moment a base role is defined. What it does is 

  install the epel repo
  install the bind-utils
  install the nfs utils 
  install the wget
  install the curl
  install yum-presto
  config domain 
  mountpoint for  /store (NFS)
  config syslog for splunk (UDP)
  install bashrc
  set motd
